# Projet docker

## Installation 

1. *Clônage du projet*

```bash 
git clone https://github.com/yanis-meziane/Projet_conteneurisation.git
```

2. *Installer toutes les dépendances*

```bash
npm install
````


## Fonctionnement 

1. *Récupération des images et lancement du projet*

```bash
docker compose up
```

2. *Adresse URL*

> Tapez dans l'URL

```bash
http://localhost:3000
````
